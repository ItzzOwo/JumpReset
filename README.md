# Owo Client

A fabric hacked client with a built in GUI, randomization, chance, self-destruct, hider and disguised as Phosphor mod. Other modules might be added in the future.

Modules:
- AutoJumpReset
- AutoSprintReset
- ForceCrash
- AutoWalk
- ForceGhostBlock
- AimAssist
- BlockInjector
- FakeGamemode
- Triggerbot
